### 关于ARM CMSIS

ARM CMSIS是一套软件接口标准，旨在简化微控制器软件开发，特别是针对ARM Cortex-M处理器家族。它包括核心外设访问层(Core Peripheral Base), DSP和RTOS接口，以及设备特定的支持包。通过使用CMSIS，开发者能够更容易地将他们的应用程序移植到不同的Cortex-M设备上，同时也能快速利用芯片特有的功能。
